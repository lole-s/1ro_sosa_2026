# Clase 03 - Hardware y software

- Duracion: 2 hs
- Unidad: U1

## Objetivo
Diferenciar hardware de software con ejemplos cotidianos.

## Inicio (10 min)
- Pregunta guia: "Se puede usar hardware sin software?"

## Desarrollo (80 min)
- Presentacion de componentes principales.
- Juego de clasificacion: hardware, software o ambos.
- Construccion de mapa conceptual por equipos.

## Cierre (20 min)
- Cada equipo comparte una conclusion.

## Con notebook
- Mapa conceptual digital.

## Sin notebook
- Mapa conceptual en afiche.

## Evidencia
Mapa conceptual con categorias correctas y ejemplos propios.
