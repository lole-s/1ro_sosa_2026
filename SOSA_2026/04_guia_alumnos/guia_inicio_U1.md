# Guia de inicio para estudiantes (U1)

- Que es una computadora y como funciona.
- Cual es la diferencia entre hardware y software.
- Para que sirven los dispositivos de entrada, salida y almacenamiento.

Escribi 3 objetos de tu casa que tengan una computadora adentro.

## Actividad 2
Clasifica estos ejemplos: teclado, parlantes, procesador de texto, camara, sistema operativo.
