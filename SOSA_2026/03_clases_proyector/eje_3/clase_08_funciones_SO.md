# Clase 08 - ¿Qué hace un Sistema Operativo?

## Objetivo
Reconocer funciones básicas del sistema operativo.

## Actividad propuesta (explícita)
1. Escribí 4 funciones del sistema operativo con ejemplo.
2. Buscá: nombre del SO, versión, RAM y almacenamiento disponible.

## Pasos
1. Definición breve.
2. Resolución de funciones.
3. Búsqueda de datos del equipo.
4. Puesta en común.

## Tiempo sugerido
- Inicio: 15 min
- Desarrollo: 50 min
- Cierre: 15 min

## Evidencia
Funciones explicadas + registro de datos.
